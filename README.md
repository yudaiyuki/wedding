# 🌸 ☀ message ☀ - メッセージシステム

メッセージシステム。QRコードでアクセスし、個人IDを入力すると封筒アニメーション付きでメッセージが表示されます。

## 🌟 特徴

- 🎊 **複数対応**: ゲスト対応メッセージシステム
- 📱 **レスポンシブデザイン**: スマホ・タブレット・PC対応
- 🎨 **夏テーマ**: 水色の空に白い雲が流れる背景
- ✨ **11種類のエフェクト**: 桜・花火・蝶・星・薔薇など
- � **封筒アニメーション**: ポストアイコン付きの封筒をタップで開封
- � **美しいフォント**: Yomogi（手書き風）で統一
- 🎴 **IDカード生成**: 各ゲスト専用のIDカードを自動生成

## 🎯 システム概要

1. **QRコード**: QRコードでアクセス
2. **ID入力**: 個人専用IDを入力
3. **封筒表示**: 教会アイコン付き封筒が表示
4. **メッセージ**: タップすると特別なメッセージとエフェクト
5. **IDカード**: 各ゲスト用の配布カード自動生成

## 📱 使用方法

1. QRコードを生成して印刷またはディスプレイに表示
2. スマートフォンのカメラでQRコードを読み取り
3. 表示されたリンクをタップ
4. ユーザーIDを入力してログイン


**各ゲストには専用のIDが割り当てられ、個別のメッセージとエフェクトが設定されています。**

## 🛠️ 技術スタック

- **フロントエンド**: HTML5, CSS3, JavaScript (ES6+)
- **認証**: フロントエンドベースの認証システム
- **アニメーション**: CSS3アニメーション（雲・エフェクト）
- **QR生成**: Python + qrcode ライブラリ
- **IDカード生成**: Python + PIL/Pillow
- **フォント**: Yomogi（手書き風）、Segoe Print、游ゴシック
- **ホスティング**: GitHub Pages対応
